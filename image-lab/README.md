# Image Lab

A privacy-first, in-browser media processing tool.

## Features

- **Pure Frontend Processing**: All image processing happens locally in your browser using Canvas API. No data is uploaded to any server.
- **Format Conversion**: Convert between PNG, JPEG, WebP, and AVIF formats.
- **Compression**: Adjust quality to reduce file size.
- **Resizing**: Scale images to specific dimensions while maintaining aspect ratio.
- **Modern UI**: Clean, responsive interface built with Vue 3 and Tailwind CSS.
- **Dark Mode**: Automatically respects system preferences.
- **Internationalization**: Support for English and Chinese.

## Tech Stack

- **Framework**: Vue 3 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v4 + Element Plus
- **State Management**: Pinia
- **I18n**: Vue I18n

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
